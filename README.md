# 🐘 Projetos em PHP

Repositório com projetos desenvolvidos durante meus estudos em PHP, com foco em desenvolvimento back-end.

## 📁 Organização

### 🔹 Projetos Simples
Exercícios voltados para a prática da linguagem, incluindo manipulação de dados, estruturas condicionais e integração básica com formulários.

### 🔹 Projetos Complexos
Projetos mais completos envolvendo lógica de back-end, organização de código, integração com banco de dados e aplicação prática dos conceitos aprendidos.

## 🚀 Objetivo

Desenvolver habilidades no desenvolvimento back-end, compreender a dinâmica entre front-end e servidor, e evoluir na criação de aplicações web utilizando PHP.
